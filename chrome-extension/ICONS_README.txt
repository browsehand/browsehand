아이콘 파일 생성 방법:

1. 온라인 아이콘 생성기 사용:
   - https://www.favicon-generator.org/
   - 👻 이모지를 업로드하거나
   - 간단한 로고 이미지 업로드

2. 필요한 크기:
   - icon16.png (16x16)
   - icon48.png (48x48)
   - icon128.png (128x128)

3. 생성된 파일을 이 폴더에 저장

임시 해결책:
- 아무 PNG 이미지를 복사해서 icon16.png, icon48.png, icon128.png로 이름 변경
- 확장 프로그램은 아이콘 없이도 작동하지만 경고가 표시됩니다
