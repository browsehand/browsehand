## 시스템 구조

```
Claude Desktop → MCP Server (localhost:8765) ↔ Chrome Extension → 브라우저 탭
```

## 빠른 시작

```bash
# MCP 서버 실행
cd mcp-server && bun run index.js

# Chrome Extension 설치
# chrome://extensions/ → 개발자 모드 → chrome-extension 폴더 로드
```
