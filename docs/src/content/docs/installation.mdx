BrowseHand를 사용하려면 **Chrome Extension**과 **MCP 서버**를 설치해야 합니다.

<Steps>

1. ## Chrome Extension 설치

   Chrome 웹 스토어에서 BrowseHand를 설치합니다.

   <Aside type="caution" title="출시 예정">
   Chrome 웹 스토어 출시 준비 중입니다. 현재는 개발자 모드로 설치해야 합니다.
   
   1. [GitHub에서 Extension 다운로드](https://github.com/user/browsehand/releases)
   2. 압축 해제
   3. Chrome에서 `chrome://extensions/` 접속
   4. "개발자 모드" 활성화
   5. "압축해제된 확장 프로그램을 로드합니다" 클릭
   6. 압축 해제한 `chrome-extension` 폴더 선택
   </Aside>

   설치 후 Chrome 우측 상단에 BrowseHand 아이콘이 나타납니다.

2. ## Claude Desktop 설정

   Claude Desktop 설정 파일을 열고 BrowseHand를 추가합니다.

   **설정 파일 위치:**
   - macOS: `~/Library/Application Support/Claude/claude_desktop_config.json`
   - Windows: `%APPDATA%\Claude\claude_desktop_config.json`

   <Tabs>
   <TabItem label="npx (추천)">
   
   가장 간단한 방법입니다. 자동으로 최신 버전을 다운로드하고 실행합니다.

   ```json
   {
     "mcpServers": {
       "browsehand": {
         "command": "npx",
         "args": ["-y", "browsehand"]
       }
     }
   }
   ```

   <Aside type="note">
   Node.js가 설치되어 있어야 합니다. [nodejs.org](https://nodejs.org/)에서 다운로드하세요.
   </Aside>

   </TabItem>
   <TabItem label="직접 다운로드">
   
   Node.js 없이 사용하려면 바이너리를 직접 다운로드하세요.

   [GitHub Releases 페이지](https://github.com/user/browsehand/releases)에서 운영체제에 맞는 파일을 다운로드:

   | 운영체제 | 파일명 |
   |----------|--------|
   | macOS (Apple Silicon) | `browsehand-macos-arm64` |
   | macOS (Intel) | `browsehand-macos-x64` |
   | Windows | `browsehand-windows-x64.exe` |
   | Linux | `browsehand-linux-x64` |

   다운로드한 파일 경로를 설정에 입력:

   ```json
   {
     "mcpServers": {
       "browsehand": {
         "command": "/Applications/browsehand"
       }
     }
   }
   ```

   <Aside type="tip">
   `/Applications/browsehand` 부분을 실제 다운로드한 파일 경로로 변경하세요.
   </Aside>

   <Aside type="note" title="macOS 사용자">
   처음 실행 시 "확인되지 않은 개발자" 경고가 나올 수 있습니다. 시스템 환경설정 → 보안 및 개인정보 보호에서 "확인 없이 열기"를 클릭하세요.
   </Aside>

   </TabItem>
   </Tabs>

3. ## Claude Desktop 재시작

   설정을 저장하고 Claude Desktop을 완전히 종료했다가 다시 실행합니다.

4. ## 연결 확인

   Chrome에서 아무 웹페이지를 열고, Claude Desktop에서 다음과 같이 말해보세요:

   ```
   연결 상태 확인해줘
   ```

   "Chrome Extension is connected" 메시지가 나오면 설치 완료입니다!

</Steps>

## 문제 해결

### Extension이 연결되지 않아요

1. Chrome에서 BrowseHand 아이콘을 클릭해 팝업을 확인하세요
2. "Connected" 상태인지 확인하세요
3. 아니라면 "Reconnect" 버튼을 클릭하세요

### Claude Desktop에서 도구가 안 보여요

1. Claude Desktop을 완전히 종료하세요 (Dock에서 우클릭 → 종료)
2. 설정 파일의 경로가 정확한지 확인하세요
3. Claude Desktop을 다시 실행하세요

### npx 실행이 안 돼요

1. Node.js가 설치되어 있는지 확인하세요: `node --version`
2. 터미널에서 `npx browsehand`를 직접 실행해 에러 메시지를 확인하세요
