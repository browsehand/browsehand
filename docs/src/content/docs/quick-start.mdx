## 기본 사용법

Chrome에서 원하는 웹페이지를 열고, Claude Desktop에서 자연어로 요청하세요.

### 페이지 내용 읽기

```
현재 페이지 내용을 읽어줘
```

```
이 페이지에서 제목만 알려줘
```

### 데이터 수집하기

```
이 페이지에서 가게 목록을 추출해줘
```

```
여기 있는 상품 이름이랑 가격을 모두 알려줘
```

### 파일로 저장하기

```
추출한 데이터를 CSV 파일로 저장해줘
```

```
이 정보들을 products.csv로 바탕화면에 저장해줘
```

## 자동화 워크플로우

Claude가 여러 작업을 순서대로 수행할 수 있습니다.

### 스크롤하며 데이터 수집

```
이 페이지를 스크롤하면서 보이는 모든 상품 정보를 수집해줘
```

### 페이지 이동하며 수집

```
네이버 지도에서 '강남역 카페'를 검색하고, 검색 결과에서 가게 이름과 평점을 50개 수집해서 CSV로 저장해줘
```

### 클릭 자동화

```
검색 결과에서 첫 번째 항목을 클릭해줘
```

```
"다음 페이지" 버튼을 클릭하고 데이터를 더 수집해줘
```

## 활용 예시

### 경쟁사 가격 조사

```
쿠팡에서 '에어팟 프로' 검색해서 상위 20개 상품의 이름과 가격을 수집해줘
```

### 부동산 시세 수집

```
직방에서 '강남구 아파트' 매물 목록을 수집해줘
```

### 맛집 리스트 만들기

```
구글 맵에서 '홍대 맛집' 검색하고, 평점 4.5 이상인 곳들의 이름, 주소, 평점을 CSV로 저장해줘
```

<Aside type="tip" title="AI가 알아서 분석해요">
복잡한 CSS 셀렉터를 몰라도 괜찮습니다. Claude가 페이지 구조를 분석해서 자동으로 데이터를 찾아냅니다.
</Aside>
