import { Card, CardGrid } from '@astrojs/starlight/components';

Claude가 여러분의 브라우저를 제어할 수 있는 기능들입니다. 복잡한 명령어를 외울 필요 없이, 자연어로 요청하면 됩니다.

## 데이터 수집

<CardGrid>
  <Card title="페이지 내용 읽기" icon="open-book">
    현재 열린 페이지의 텍스트를 읽어옵니다.
    
    *"이 페이지 내용 읽어줘"*
  </Card>
  <Card title="구조화된 데이터 추출" icon="list-format">
    상품 목록, 가게 정보 등 반복되는 데이터를 표 형태로 추출합니다.
    
    *"여기서 상품명이랑 가격 추출해줘"*
  </Card>
</CardGrid>

## 브라우저 조작

<CardGrid>
  <Card title="클릭" icon="rocket">
    버튼이나 링크를 클릭합니다.
    
    *"검색 버튼 클릭해줘"*
  </Card>
  <Card title="스크롤" icon="down-caret">
    페이지를 위아래로 스크롤합니다.
    
    *"페이지 맨 아래까지 스크롤해줘"*
  </Card>
  <Card title="페이지 이동" icon="external">
    다른 URL로 이동합니다.
    
    *"네이버로 이동해줘"*
  </Card>
  <Card title="대기" icon="seti:clock">
    특정 요소가 나타날 때까지 기다립니다.
    
    *"로딩이 끝날 때까지 기다려줘"*
  </Card>
</CardGrid>

## 파일 저장

<CardGrid>
  <Card title="CSV 저장" icon="document">
    수집한 데이터를 엑셀에서 열 수 있는 CSV 파일로 저장합니다.
    
    *"이 데이터를 products.csv로 저장해줘"*
  </Card>
  <Card title="JSON 저장" icon="seti:json">
    데이터를 JSON 형식으로 저장합니다.
    
    *"data.json으로 저장해줘"*
  </Card>
</CardGrid>

## 고급 기능

<CardGrid>
  <Card title="페이지 분석" icon="magnifier">
    Claude가 페이지 구조를 분석해서 어떤 데이터를 추출할 수 있는지 파악합니다.
    
    *"이 페이지 구조 분석해줘"*
  </Card>
  <Card title="스크립트 실행" icon="setting">
    JavaScript 코드를 실행합니다. (고급 사용자용)
    
    *"document.title 값 알려줘"*
  </Card>
</CardGrid>
