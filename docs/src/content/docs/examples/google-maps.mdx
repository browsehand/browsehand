구글 맵에서 원하는 지역의 맛집이나 카페 정보를 수집하는 예제입니다.

## 한 번에 요청하기

Claude에게 이렇게 말해보세요:

```
구글 맵에서 '강남 카페' 검색해서 50개 가게의 이름, 평점, 주소를 수집하고 CSV로 저장해줘
```

Claude가 알아서:
1. 구글 맵으로 이동
2. '강남 카페' 검색
3. 결과를 스크롤하면서 데이터 수집
4. CSV 파일로 저장

## 단계별로 요청하기

<Steps>

1. **구글 맵 열기**
   
   Chrome에서 구글 맵(maps.google.com)을 열어주세요.

2. **검색하기**
   
   ```
   강남 카페 검색해줘
   ```

3. **스크롤하며 더 많은 결과 보기**
   
   ```
   검색 결과를 5번 스크롤해줘
   ```

4. **데이터 추출하기**
   
   ```
   검색 결과에서 가게 이름, 평점, 주소를 추출해줘
   ```

5. **파일로 저장하기**
   
   ```
   추출한 데이터를 gangnam-cafes.csv로 저장해줘
   ```

</Steps>

## 결과 예시

바탕화면에 저장된 CSV 파일을 엑셀로 열면:

| 이름 | 평점 | 주소 |
|------|------|------|
| 스타벅스 강남점 | 4.3 | 서울 강남구 테헤란로... |
| 블루보틀 강남 | 4.5 | 서울 강남구 강남대로... |
| 카페 드 파리 | 4.2 | 서울 강남구 역삼동... |
| ... | ... | ... |

<Aside type="tip">
평점이 높은 가게만 수집하고 싶다면: "평점 4.5 이상인 카페만 수집해줘"
</Aside>
