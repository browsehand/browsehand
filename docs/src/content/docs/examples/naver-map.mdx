네이버 지도에서 맛집, 카페, 병원 등 다양한 가게 정보를 수집하는 예제입니다.

## 한 번에 요청하기

Claude에게 이렇게 말해보세요:

```
네이버 지도에서 '홍대 맛집' 검색해서 가게 정보를 30개 수집하고 CSV로 저장해줘
```

## 단계별로 요청하기

<Steps>

1. **네이버 지도 열기**
   
   Chrome에서 네이버 지도(map.naver.com)를 열어주세요.

2. **검색하기**
   
   ```
   홍대 맛집 검색해줘
   ```

3. **결과 스크롤**
   
   ```
   검색 결과를 쭉 스크롤해줘
   ```

4. **데이터 추출**
   
   ```
   가게 이름, 카테고리, 주소, 전화번호를 추출해줘
   ```

5. **저장**
   
   ```
   hongdae-restaurants.csv로 저장해줘
   ```

</Steps>

## 결과 예시

| 이름 | 카테고리 | 주소 | 전화번호 |
|------|----------|------|----------|
| 연남동 돈까스 | 일식 | 서울 마포구 연남동... | 02-123-4567 |
| 홍대 칼국수 | 한식 | 서울 마포구 서교동... | 02-234-5678 |
| 망원 피자 | 양식 | 서울 마포구 망원동... | 02-345-6789 |

## 다른 활용 예시

### 병원 찾기
```
네이버 지도에서 '강남역 피부과' 검색해서 병원 정보 수집해줘
```

### 부동산 정보
```
네이버 지도에서 '판교 아파트' 검색해서 매물 정보 수집해줘
```

### 주변 편의시설
```
네이버 지도에서 '역삼역 편의점' 검색해서 위치 정보 수집해줘
```

<Aside type="note">
네이버 지도는 일부 정보가 iframe으로 되어 있어 수집이 안 되는 경우가 있습니다. 이 경우 다른 검색 방법을 시도해보세요.
</Aside>
